## Project Overview

  

> The Student Store is a non-profit marketplace for students in Yemen to sell, buy, or borrow affordable used products like electronics, games, and study books. It aims to provide essential eCommerce experience and support for students.

  

>

  

> Service providers and resellers can also offer their products and services at low prices, with a custom blog on the main website providing training materials for users. The platform includes sign-in and sign-up features for users to order products or services.

  

>

  

> This project was developed by the CodeForce team as part of the Re:Coded Frontend Web Development Bootcamp in Yemen.

  

<br>

  

## Project Features

  

-  **Authentication:** Users can sign in and sign up to the platform using their email address and password. Users can also sign in using their Google account.

  

-  **Internationalization:** The platform supports both English and Arabic languages.

  

-  **Responsive Design:** The platform is responsive and can be viewed on mobile, tablet, and desktop devices.

  

-  **Product Categories:** Products are organized into categories to make it easier for users to find what they are looking for.

  

-  **Product Search:** Users can search for products by name or category.

  

-  **Product Details:** Users can view the details of a product, including its name, description, price, and seller.

## Installation

  

```shell

npm install  -g  commitizen  //  Installs  Commitizen  globally (Commitizen is  used  to  create  Conventional  Commits)

  

npm install  //  Installs  the  Node  modules  required  for  the  project  to  run (this command  is  run  in  the  root  directory  of  the  project)

  

npx husky  install  //  Installs  Husky  if  it  is  not  already  installed (Husky is  used  to  run  the  Prettier  and  ESLint  tools  before  committing  changes)

  

chmod ug+x  .husky/*  //  Gives  Husky  executable  extension  permissions (this command  is  run  in  the  root  directory  of  the  project)

```

  

## Development Process and Workflow

  

To contribute to this project, follow the steps below:

  

1. Fork the repository and clone it to your local machine.

  

```bash

git clone  https://github.com/your-username/student-store.git

```

  

2. Install the necessary dependencies by running the following command in the root directory of the project:

  

```bash

npm install

```

  

3. Create a new branch with a descriptive name for the feature or bug you are working on:

  

```bash

git checkout  -b <branch-name>

```

  

4. Make your changes to the code and commit them using the Commitizen tool and Conventional Commits. Use the following command to commit your changes:

  

```bash

git add  . && git cz

```

  

This will prompt you to fill in the necessary details for your commit message.

  

5. Push your changes to your forked repository:

  

```bash

git push  origin <branch-name>

```

  

6. Submit a pull request to the main repository. Make sure to describe your changes clearly and provide any necessary context. Once your pull request is approved, your changes will be merged into the main branch.

  

Note: Before committing your changes, make sure to run the Prettier and ESLint tools to ensure that your code follows the project's coding style guidelines. You can run these tools using the following commands:

  

```bash

npm run  prettier

  

npm run  lint

```

## Project Prototype

  

1. Homepage: The homepage displays a logo image, a search bar, and a list of product categories, such as books, furniture, electronics, and two-wheels. Each category has a thumbnail and price sorting.

  

2. Login page: The login page allows users to enter their email address and password to access their accounts. Users can also log in with their Google, Facebook, or Twitter accounts.

  

3. Registration page: The registration page allows new users to create an account by providing their name, email address, password, and address. Users can also create an account using their Google, Facebook, or Twitter accounts.

  

4. Single product page: Clicking on a product takes the user to a page that displays detailed information about the product, including a larger and more image, a longer description, and a product title. Users can add the product to the cart.

  

5. Product Listing Page: The Product Listing page allows users to create a new product list by providing product details, such as name, description, price, and photo. Users can also select the product category and locate the product.

  

6. Donation page: The donation page allows users to make a donation to support the Student Store. Users can select the amount they wish to donate and select a payment method.

  

7. My Orders page: The My Orders page allows users to view their order history, including order date and price.

  

8. My Lists page: The My Lists page allows users to view their product listings, including product names, categories, and prices. Users can also edit or delete their listings.

  

9. Edit profile page: The Edit profile page allows users to update their account information, such as name, email address, password, and address.
